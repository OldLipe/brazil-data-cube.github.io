Search¶

Search Features¶

  • Choose from which resources you want to get the data, between Data Cubes, Collections, Classifications and Mosaics (groups on the left side of your screen);

  • Choose from which collection you want to receive the data (check boxes inside the resources);

  • Filter the collection by using the location. You can do it either by drawing on the map with the select region button or enter a location in the search field on the right tab of the location component;

  • Filter the collection by using the time range by either typing the date or select the date from calendar;

Additional Features¶

  • You can see the full name of a collection by hovering mouse over the collection name;

  • When selecting a collection, clicking on box icon, the time range fields will be filled (You can either use the full range or change);

  • Select Date by either typing the date or select the date from calendar;
