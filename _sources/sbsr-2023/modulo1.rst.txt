Manhã
-----

Nesse período será feita uma introdução sobre séries temporais e cubos de dado, serão apresentados os dados do projeto *Brazil Data Cube (BDC)*, bem como o portal *BDC Explorer* e serviço de coleta amostras *TerraCollect*, além de utilizar os serviços *SpatioTemporal Asset Catalog (STAC)*, *Web Time Series Service (WTSS)* e *Web Land Trajectory Service (WLTS)*, em linguagem de programação R.

Os códigos para acompanhamento do curso estão disponíveis na plataforma Kaggle nos links abaixo, mas os arquivos originais são mantidos e atualizados no repositório *Code-Gallery* no github do projeto (https://github.com/brazil-data-cube/code-gallery).
