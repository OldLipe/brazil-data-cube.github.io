2° Módulo
---------

Nesse módulo os dados do projeto Brazil Data Cube (BDC) serão utilizando, por meio da linguagem R, no pacote de classificação e análise de séries temporais SITS.

Os códigos para acompanhamento do curso estão disponíveis na plataforma Kaggle nos links abaixo. Os notebooks originais e que são mantidos atualizados encontram-se no repositório Code-Gallery no github do projeto (https://github.com/brazil-data-cube/code-gallery).
